Izveidojiet jaunu Laravel aplikāciju 'products'.
1. Versionējiet to un nosūtiet uz savu Github kontu ar nosaukumu 'products-06032026'.
2. Pievienojiet aplikācijai jaunu maršrutu, kas saņem lietotāja ceļu '/about' un apstrādā to ar anonīmu funkciju izvadot 'h1' elementu ar tekstu "About this page" bez skata (view).
3. Versionējiet izmaiņas.
4. Izveidojiet jaunu maršrutu ar ceļu '/info', kas atgriež skatu 'info'. Izveidojiet šo skatu un ievietojiet tajā statisku paragrāfa elementu, kurā ir 25 vārdus garš 'lorem' gadījuma teksts.
5. Versionējiet izmaiņas.
6. Izveidojiet arī jaunu modeli 'Product' ar migrāciju, kas izveido jaunu datubāzes tabulu ar vienu 'string' tipa lauku 'name'. Veiciet migrāciju un izveidotajā tabulā manuāli ievietojiet divus testa ierakstus.
7. Versionējiet izmaiņas.
8. Izveidojiet skatu, kas attēlo visus produktus no datubāzēs. Šajā gadījumā tikai katra produkta nosaukumu, jo citu lauku nav. Attēlojiet nosaukumu kā 'h1' elementu.
9. Izveidojiet kontrolieri un metodi, kas atgriež skatu attēlojot lietotājam visus produktus.
10. Izveidojiet maršrutu, kas apstrādā '/products' ceļu GET pieprasījumam un servē lietotājam visus produktu sarakstu.
11. Versionējiet izmaiņas.
12. Atsūtiet skolotājam Github saiti privātā sarakstē.
